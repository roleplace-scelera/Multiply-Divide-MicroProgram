NAME
       mic1sim - mic1 simulator


SYNOPSIS
       java mic1sim


DESCRIPTION
       mic1sim  simulates  the Mic-1 microprogrammed architecture
       as described in Chapter 4 of Andrew S.  Tanenbaum,  Struc-
       tured Computer Organization, 4e, (Prentice-Hall, 1998).


       mic1sim  provides  a means for step-by-step execution of a
       Mic-1 microprogram in the interpretation of an instruction
       set architecture (ISA-level) macroprogram.





ENVIRONMENT
       The mic1 file classes.zip must be in your CLASSPATH.
